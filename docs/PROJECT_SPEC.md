## Intent

BlackstoneFinance.webflow is the public-facing website for PortfolioEngine, a portfolio management and investment analysis platform. It is a Webflow export (HTML5, CSS3, vanilla JavaScript) hosted on GitHub Pages at conjur.pro. The site serves as the marketing and product showcase for the PortfolioEngine financial intelligence service. An executing agent should optimize for clean professional presentation and correct financial services messaging. The site was originally a Webflow portfolio template ("Jane Lo") that is being transformed into a PortfolioEngine product marketing site with a premium dark theme.

---

## Core Constraints

- MUST maintain Webflow export compatibility — no custom build tools or frameworks
- MUST follow financial services professional standards for content and design
- MUST NOT include client data or real portfolio information
- MUST NOT add framework dependencies (React, Vue, etc.) — vanilla HTML/CSS/JS only
- MUST use the Webflow MCP tools from the `mcp__claude_ai_Webflow__*` set (not `mcp__webflow__*`)
- MUST keep Designer tab active and in foreground when using Designer API calls
- MUST batch Designer API actions to 2-3 per call to avoid timeouts
- PREFER editing via Webflow Designer API over direct HTML file edits (keeps Webflow project in sync)

---

## Quality Attributes

- Professional presentation over technical sophistication — this is a marketing site, not an app
- Content accuracy over visual flair — financial services messaging must be credible and precise
- Mobile responsiveness over desktop-first features
- Webflow compatibility over hand-coded optimizations — edits should round-trip through Webflow

## Domain Context

### Terminology
- **Webflow export**: A static HTML/CSS/JS site generated by Webflow's visual designer. The exported files can be hosted anywhere but should remain compatible with Webflow for future editing.
- **Designer API**: Webflow's live design tool API, accessed through MCP tools. Requires the Webflow Designer tab to be open and active in Chrome. Used for text updates and style changes.
- **Data API**: Webflow's REST API for site metadata, SEO, and page settings. Does not require Designer to be open.
- **Component**: A reusable element in Webflow (Navigation, Contact, Footer). Editing component text requires editing the component definition, not page instances.
- **PortfolioEngine**: The financial product being marketed. Features include portfolio dashboards, thesis-based investing, risk analytics, and client reporting.
